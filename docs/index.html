C4N防災マップ

このマップについて

防災に関して、地域で気がついたことをみんなで投稿していき、共有する防災マップです。防災マップを片手に、フィールドワークを行い、写真を撮って、投稿していましょう！

インストール方法

PWAに対応しているので、WEBブラウザ（Safari、Google Chromeなど）の「ホーム画面に追加」からインストールできます。
